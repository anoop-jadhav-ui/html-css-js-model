# Skills 3D Models

Elevate your projects with stunning 3D icons for various technical skills.

## Steps to run the app locally

1. Install node modules

```
yarn
```

2. Run app

```
yarn dev
```

## 3D Model

As of now I have created models for following skills -

- HTML
- CSS
- JavaScript
- Figma
- React
- Blender
